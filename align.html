Left/Top Alignment how do we use it?

Understanding how to align your divs you need to have a basic understanding of CSS (Cascading Style Sheets). Without CSS, HTML would would have no aesthetic. CSS allows us to bring out HTML to life by adding specific properties to each of our divs.

There are a few ways of implementing CSS. The first way uses identifiers, like grav and solid, to apply the properties to your divs. There are two popular types of identifiers used for CSS. class= and id=. Classes can have multiple identifiers strung together where as id can only use one. We won't get into convention and aesthetic of css in this tutorial.

you can call on these different types of identifiers with a .css file or within your HTML document. For this example we will simply use the second method. Inspect the code below.

					
<style>
.left-top{
	position: absolute;
	width: 50px;
	height: 50px;
	background: red;
	left: 0px;
	top: 0px;
}
.right-bottom{
	position: absolute;
	width: 50px;
	height: 50px;
	background: red;
	right: 0px;
	bottom: 0px;
}
</style>

You can also insert the CSS files into the div itself by style="code here". 

<div class="middle" style="position: absolute; width: 50px; height: 50px; left: 50%; bottom: 50%;">1</div>
By default, divs are aligned statically. Applying position: absolute; will break out of this and will break the grid of the divs. With position: absolute;, you can now use top:number; bottom:number; left:number; right:number; to place the div around the page as you please.
