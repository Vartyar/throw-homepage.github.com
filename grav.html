Gravity Identifier how do we use it?

There are a few important things to keep in mind when it comes to bringing your project to life. In order to get our code working we need to use identifiers that will attach to objects in your HTML document. We have provided you with a few identifiers, each with unique attributes, to get you started. The first, and most important, is our gravity tag.

Inspect the input code we have provided you on the right side of the screen and see if you can pinpoint how the gravity tag has been attached to the different objects. You've most likely noticed that we used a value called class="grav" on each of the elements that are now slave to our gravity enviornment! This tag is the only way in which our objects can have the specific attributes attached to grav.

The class="grav" needs to be put inside the of our HTML object.

BAD! 
<span style="top:200px;left:400px; background: black"> class="grav"</span>
GOOD! 
<span  class="grav" style="top:200px;left:400px;  background: black"></span>

Give it a try on the code provided on the right side of the screen. Remember, if you reload the page all work will be lost. :(
